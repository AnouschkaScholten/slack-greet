Code for NL is de Nederlandse community van developers en designers die samenwerken aan een open, eerlijke, en inclusieve digitale overheid en samenleving. Ons doel is een sterke Nederlandse _public tech_ beweging die vanuit publieke waarden en belangen werkt aan de digitale transformatie van overheid en maatschappij. Code for NL is onderdeel van het internationale Code for All netwerk. Voor meer informatie, zie www.codefor.nl

*Wat moet je weten voor je begint?*

* Op deze Slack omgeving geldt de Code for NL gedragscode (https://github.com/codefornl/gedragscode). 
* Voor vragen en opmerkingen is het bestuur te vinden op <#CDBR8Q56W|#bestuur-publiek>.
* Code for NL geloof in een sterke _public tech_ beweging die werkt vanuit Community Principles (https://github.com/codefornl/community-principes)
* Wij onderschrijven de Standard for Public Code (https://standard.publiccode.net/)
* Code for NL is een omgeving waar 'doen' en 'open' centraal staat.
